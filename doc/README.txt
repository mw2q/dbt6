OSDL-DBT-3 are OpenOffice files.
This files gives an overview of the documentations. 
==================================================
dbt6-architecture.sxw: gives an overview of OSDL-DBT-3 and a brief description of TPC-H

dbt6-tpch.sxw: lists the places where the OSDL-DBT-3 deviates from TPC-H

dbt6-user-manual.sxw:  provides instructions on how to set up and use the OSDL-DBT-3 test kit.  Suggestions how to tune the SAPDB parameters are also discussed.

dbt6-maintain.sxw: explains the structure of OSDL-DBT-3 and its implementation
